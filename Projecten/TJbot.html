TJbot Speech Recognition Project

Project Overview

Working with the secretary of Roda, we developed a TJbot-based solution to address racism awareness. Using Node-RED and Raspberry Pi, we created a speech-to-text system integrated with Google Cloud Services to facilitate meaningful conversations about racism.
